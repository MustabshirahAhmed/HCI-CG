Norman’s Model vs GOMS and KLM – ATM Withdrawal
Norman’s Stage ATM Example (User Perspective) GOMS Interpretation (Task Structure) KLM Interpretation (Low-level Actions) Analysis (Model Alignment)
Goal User forms a clear goal: “I need Rs. 5000 cash for immediate use.” GOAL: Withdraw cash.
This is the top-level objective defined before any action starts.		 Not modeled. KLM assumes the goal is already fixed. Direct Match – Norman’s Goal corresponds exactly to GOMS Goal.
Plan / Intention User decides: “I’ll use the ATM nearby instead of going inside the bank.” METHOD: ATM withdrawal method.
Selection Rule: Choose ATM because it is faster and convenient.		 Not modeled. KLM does not represent planning decisions. Strong Match – Norman’s planning maps to GOMS Method and Selection Rule.
Specify User mentally plans the steps: insert card → enter PIN → select withdrawal → enter amount. Assumed to be already defined inside the chosen method. No explicit cognitive step is shown. May involve a Mental operator [M], but only as preparation, not detailed reasoning. Partial Match – Norman explicitly models thinking; GOMS and KLM largely abstract it.
Execute User physically inserts card, types PIN, presses screen buttons. OPERATORS: Key presses, button selections, card insertion. [M][P][K] sequence: [M] prepare → [P] point → [K] press. Direct Match – All three models represent physical execution clearly.
Perceive User sees screen message “Processing…” and hears ATM sounds. Not modeled. Correct perception is assumed. Not modeled. No perception stage exists. Major Difference – Only Norman explicitly includes perception.
Interpret User understands: “The ATM is checking my PIN and balance.” Assumed perfect and automatic understanding. Ignored completely. Major Difference – Interpretation is central to Norman but absent in GOMS/KLM.
Evaluate Cash is dispensed; user checks amount and confirms goal is achieved. Task is assumed complete once operators finish successfully. Task ends when last operator finishes; no evaluation. Key Difference – Only Norman checks whether the goal was truly achieved.
